## Frequently Asked Questions

Here are answers to common questions about using FormBold on HTML websites.

1. **How do I add a contact form to a plain HTML website?**

	Use FormBold’s API endpoint as your form action and set the method to POST. No JavaScript or backend required.

2. **Can I receive form submissions directly to my email?**

	Yes, submissions are instantly delivered to your inbox and any connected third-party apps.

3. **Is this secure? Are submissions encrypted?**

	Absolutely. All submissions are end-to-end encrypted and handled over secure channels.

4. **Can users upload files through the HTML form?**

	Yes. FormBold supports file uploads and provides a CDN-hosted download link with each submission.

5. **Do I need to write any backend code?**

	No backend coding is required—just copy the endpoint and drop it into your form.

6. **Is it free to use?**

	Yes. FormBold offers a generous free tier suitable for personal, hobby, and early-stage projects.
