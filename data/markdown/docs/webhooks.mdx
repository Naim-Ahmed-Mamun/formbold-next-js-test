## About webhooks

Webhooks allow you to receive real-time HTTP notifications of your form submissions. 

## Webhook format

```json
{
  "payload": {
    "field1": "value1",
    "field2": "value2",
    "field3": "value3",
    "etc": "as many fields as your submission has...",
  },
  "files": {
    
  }
}
```
