Being spammed by bots one of the biggest issue while using a form on websites,
thats why we have multiple options to protect your form from spammers.

**Filter Spam:**
Automatic spam filtering option that actually uses our essential server side mechanism to defend spam submissions, you can turn it on for basic spam filtering.

reCAPTCHA is a google service that provides strict spam protection. If you enable you have to put [reCAPTCHA](https://www.google.com/recaptcha) site key and secrete key to make it work.
Highly recommended for strict spam protection.
